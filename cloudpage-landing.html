CloudPage Landing Page

Salesforce Marketing Cloud CloudPages Concept Project

Project Overview

This project demonstrates a responsive landing page concept inspired by Salesforce Marketing Cloud CloudPages. The page is designed to capture subscriber preferences and support personalized lifecycle marketing campaigns.

Landing Page Mockup

Hello %%FirstName%%, choose your preferences

Tell us what types of updates you’d like to receive so we can personalize your experience.

Personalization Example

This mockup uses an AMPscript-style personalization string:

Hello %%FirstName%%

In a real Salesforce Marketing Cloud CloudPage, subscriber attributes could be used to personalize content and update Data Extension records.

SFMC Concepts Demonstrated

  • CloudPages
  • Subscriber Preferences
  • Data Extension Updates
  • Personalization Strings
  • AMPscript Concepts
  • Responsive HTML/CSS
  • Consent and Preference Management
